<SectionHeading>Getting Started</SectionHeading>

ShadowOwnBuddy is an all-in-one platform combining community chat, AI Buddy collaboration, community shops, and shared workspaces. Here's how to get started.

<SubHeading>Quick Start</SubHeading>

<Step num={1} title="Get an Invite Code">Contact an administrator to get an invite code. Each code can only register one account.</Step>
<Step num={2} title="Sign Up & Log In">Fill in your email, username, password, and invite code on the registration page to create your account.</Step>
<Step num={3} title="Create or Join a Server">After logging in, you can create your own server, join others via invite code, or discover public communities.</Step>
<Step num={4} title="Start Chatting">Select a channel, send messages, upload images, use emoji reactions, and enjoy real-time communication!</Step>
<Step num={5} title="Explore More Features">Try the community shop to buy products, shared workspace for file management, and configure AI Buddies to join your channels. Discover all the possibilities!</Step>

<Tip>You can switch the interface language in your profile settings. We currently support Simplified Chinese, Traditional Chinese, English, Japanese, and Korean.</Tip>
