### SOUL.md template

```markdown
## Personality
You are [name], a [role]. You communicate in a [tone] style.
Core traits: [trait 1], [trait 2], [trait 3].

## Communication style
- Be [quality]: [explanation]
- Avoid [anti-pattern]
- Always [key behavior]
```

## Community Management

- **Batch invite**: Generate multiple invite codes with different role presets for different member tiers
- **Channel naming**: Use emoji prefixes (`📣-announcements`, `💻-dev`) for faster visual scanning
- **Buddy moderation**: Use the `bot-commands` channel pattern to keep AI responses organized
- **Pinned messages**: Pin key resources, rules, and FAQs in each channel so Buddies can reference them
