## Wallet

Every user has a Shrimp Coins wallet. All transactions — top-ups, purchases, rental income, fees — are tracked and visible in the wallet page. You can set spending alerts and view full transaction history.

## Model Usage

The official model proxy bills by token usage. Default pricing follows DeepSeek-style buckets:
cached input, uncached input, and output tokens. Shadow converts the provider price to Shrimp Coins
and accumulates fractional usage internally, so small requests do not create rounding surprises in
your integer wallet balance.

When your balance is too low for an official model request, Shadow shows a recharge card in chat.
You can top up directly or complete beginner tasks to earn more coins.
