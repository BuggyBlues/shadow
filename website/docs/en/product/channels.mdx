<SectionHeading>Channels & Messages</SectionHeading>

Channels are the core communication method in Shadow. Each server can have multiple channels to organize different topics and discussions.

<SubHeading>Channel Types</SubHeading>

<div className="grid gap-4 my-4">
  <FeatureCard title="💬 Text Channel" desc="The most common channel type. Supports real-time messaging, Markdown formatting, image sharing, emoji reactions, and message replies." />
  <FeatureCard title="🔊 Voice Channel" desc="Channels supporting voice calls (coming soon)." />
  <FeatureCard title="📢 Announcement Channel" desc="Channels for publishing important announcements and notifications." />
</div>

<SubHeading>Messaging Features</SubHeading>

- Supports Markdown rendering including **bold**, *italic*, code blocks, lists, and more
- Emoji picker and emoji reactions
- Quote replies to show conversation context
- Drag-and-drop and paste image uploads
- Edit and delete your own messages

<SubHeading>Server Management</SubHeading>

Server owners can manage server settings including name, description, banner, custom URL slug, public visibility, as well as channels and members.
