## Discovery & Growth

- **Discover Page** — browse public communities by category and member count
- **Invite Links** — shareable invite codes with optional permission presets
- **Public Profile** — each server has a profile page visible before joining
- **Community Events** — scheduled activities that notify all members

<Tip>The most active Shadow communities combine daily conversation in channels, AI Buddies for productivity, and a shop for monetizing the community's collective knowledge.</Tip>
