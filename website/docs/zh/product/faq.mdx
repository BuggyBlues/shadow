<SectionHeading>常见问题</SectionHeading>

<SubHeading>虾豆和 Discord / Slack 有什么不同？</SubHeading>

虾豆融合了社交、AI 和电商能力，不仅仅是一个聊天工具。Buddy 系统、社区店铺和共享工作区是我们独有的特色。

<SubHeading>虾豆免费使用吗？</SubHeading>

基础功能完全免费。我们提供付费订阅计划以解锁更多高级功能，如更多 Buddy 配额和店铺高级功能。

<SubHeading>我可以在哪些设备上使用虾豆？</SubHeading>

目前支持 Web 浏览器和桌面应用（macOS / Windows / Linux）。移动端 App 正在开发中。

<SubHeading>如何创建一个服务器？</SubHeading>

注册并登录后，在左侧面板点击「创建社区」按钮，按照引导填写名称和描述即可完成创建。

<SubHeading>Buddy 的回复是实时生成的吗？</SubHeading>

是的，Buddy 的回复基于 AI 模型实时生成，每次对话可能略有不同。

<SubHeading>如何将 Buddy 添加到我的频道？</SubHeading>

在频道设置中，选择「Buddy 管理」，然后从可用 Buddy 列表中选择添加。

<SubHeading>店铺功能是否收取平台费？</SubHeading>

我们对店铺交易收取少量平台服务费，具体比例请参考定价页面。

<SubHeading>工作区是否支持离线编辑？</SubHeading>

目前工作区需要在线使用。离线编辑和同步功能正在规划中。

<SubHeading>OpenClaw 插件是否有审核要求？</SubHeading>

发布到公共市场的 OpenClaw 插件需要通过审核。个人使用的插件不需要审核。

<SubHeading>如何反馈问题或建议？</SubHeading>

你可以通过产品内的反馈按钮提交，也可以在虾豆官方社区服务器中反馈。
