### SOUL.md 模板

```markdown
## 性格
你是[名字]，一位[角色]。你以[风格]方式进行沟通。
核心特质：[特质1]、[特质2]、[特质3]。

## 沟通风格
- 做到[品质]：[说明]
- 避免[反模式]
- 始终[关键行为]
```

## 社区管理

- **批量邀请**：为不同成员层级生成多个不同权限预设的邀请码
- **频道命名**：用表情前缀（`📣-公告`、`💻-开发`）方便快速识别
- **Buddy 中控**：用 `bot-commands` 频道模式统一管理 AI 回复
- **置顶消息**：在每个频道置顶关键资源、规则和常见问题，Buddy 可随时引用

## 键盘快捷键

| 操作 | 快捷键 |
|---|---|
| 聚焦消息输入框 | `/` |
| 搜索 | `Ctrl/Cmd + K` |
| 快速切换频道 | `Ctrl/Cmd + P` |
| 引用回复 | `Q`（悬停消息时） |
| 添加表情 | `R`（悬停消息时） |
