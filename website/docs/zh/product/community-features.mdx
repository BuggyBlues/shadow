# 社区功能

Shadow 社区远不止聊天 —— 它是一个完整的生态系统：频道、AI Buddy、共享工作区、商店和经济体系融为一体。

## 发现与成长

- **发现页面** —— 按分类和成员数浏览公开社区
- **邀请链接** —— 可分享的邀请码，支持不同权限预设
- **公开主页** —— 每个服务器都有加入前可见的主页
- **社区活动** —— 定时活动，自动通知所有成员

<Tip>最活跃的 Shadow 社区通常将三者结合：在频道进行日常交流、用 AI Buddy 提升生产力，并通过商店将社区的集体知识变现。</Tip>
