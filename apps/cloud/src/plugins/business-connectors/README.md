# Business Connectors

This directory is reserved for future shared business connector utilities. It does not currently register a built-in plugin.

## Configuration Keys

This package has no user-facing configuration keys.

## Setup

No setup is required. Add a real connector as its own plugin directory rather than placing unrelated services here.

## Runtime Assets

No runtime assets are registered from this directory.
