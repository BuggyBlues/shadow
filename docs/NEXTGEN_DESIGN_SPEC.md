## 1. 品牌定位

### 1.1 核心定位

**Shadow = 创作者的数字分身生态**

从"社区工具"升级为"创作者的 AI 原生社交空间"。

```
旧定位：The super community for super individuals.
新定位：你的 AI 分身，你的社区，你的世界。
```

### 1.2 目标用户画像

| 维度 | 描述 |
|------|------|
| **年龄** | 18-28 岁（Z世代核心）|
| **身份** | 内容创作者、独立开发者、Web3 玩家、AI 原住民 |
| **特质** | 重视个性表达、反感说教、喜欢"搞怪"但有边界 |
| **痛点** | 微信太正式、Discord 太复杂、Telegram 太工具化 |
| **需求** | 一个能承载 AI 分身、能赚钱、能社交的"数字家园" |

### 1.3 品牌人格

| 特质 | 表达 |
|------|------|
| **真实** | 不装、不端着、允许"翻车" |
| **好玩** | 彩蛋、梗、随机惊喜 |
| **赋能** | 让创作者赚钱、让 AI 有用 |
| **开放** | 开源、可定制、社区驱动 |

### 1.4 品牌口号方案

```
主推：Shadow — 你的数字分身世界
备选 1：不只是聊天，是生活
备选 2：AI + 你 = 新物种
备选 3：在这里，AI 是你的队友
```

### 1.5 品牌声音

**Do:**
- 轻松、有梗、不说教
- 用"你"而不是"您"
- 适度玩梗、适度自嘲
- 强调用户的价值和创造力

**Don't:**
- 不要用企业腔、公关腔
- 不要过度承诺
- 不要用"革命性"、"颠覆"这种大词

### 2.7 动效规范

#### 时长标准

```css
--duration-instant:  100ms;  /* 微交互：按钮按下 */
--duration-fast:     150ms;  /* 状态切换：hover */
--duration-normal:   250ms;  /* 常规过渡：展开/收起 */
--duration-slow:     400ms;  /* 页面切换、模态框 */
--duration-slower:   600ms;  /* 大动作：抽屉、全屏 */
```

#### 缓动曲线

```css
--ease-out:      cubic-bezier(0.16, 1, 0.3, 1);    /* 退出、消失 */
--ease-in:       cubic-bezier(0.7, 0, 0.84, 0);    /* 进入、出现 */
--ease-in-out:   cubic-bezier(0.65, 0, 0.35, 1);    /* 双向过渡 */
--ease-spring:   cubic-bezier(0.34, 1.56, 0.64, 1); /* 弹性效果 */
```

#### 动效原则

1. **响应即时** — 点击反馈 ≤ 100ms
2. **节奏明快** — 总时长尽量 ≤ 400ms
3. **意图清晰** — 动效服务于功能，不为炫而炫
4. **可关闭** — 尊重 `prefers-reduced-motion`

### 3.2 组件规范

#### Button 按钮

```tsx
// 变体
type ButtonVariant = 
  | 'primary'    // 品牌渐变填充
  | 'secondary'  // 中性色填充
  | 'outline'    // 描边透明
  | 'ghost'      // 无背景
  | 'danger';    // 红色警示

// 尺寸
type ButtonSize = 'sm' | 'md' | 'lg' | 'xl';

// 状态
- Default: 基础样式
- Hover: 背景加深 10%，shadow-glow
- Active: 缩放 0.98，背景加深 20%
- Disabled: opacity 0.5, cursor not-allowed
- Loading: spinner + 文字变灰
```

#### Input 输入框

```tsx
// 变体
type InputVariant = 'default' | 'filled';

// 尺寸
type InputSize = 'sm' | 'md' | 'lg';

- Focus: primary-500 边框, shadow-glow

- Disabled: opacity 0.5

// 增强
- 支持前后图标/按钮
- 支持浮动标签
- 支持字符计数

### 3.3 导航模式

#### 服务器列表（Server Sidebar）

```
设计原则：
- 图标优先：服务器以图标/emoji展示，名称隐藏
- 分隔清晰：文件夹/直接加入的服务器用间距区分
- 快捷操作：悬停显示快速操作（设置、通知）
- 创建入口：底部固定"+"按钮

视觉：
- 尺寸: 48x48px 图标 + 8px 间距
- 激活态: 左侧 4px 品牌色竖条 + 背景高亮
- 悬停态: 背景轻微高亮
```

#### 频道列表（Channel List）

```
设计原则：
- 层级清晰: 分类 > 频道 > 线程
- 信息密度: 文字频道 # / 语音频道 🔊 / 直播 📹
- 未读状态: 白点 + 加粗文字
- 提及状态: 品牌色圆点 + 加粗

交互：
- 滑动操作: 右滑显示快速操作（静音、设置）
- 长按: 显示上下文菜单
- 拖拽: 支持频道排序（需权限）
```

### 3.4 聊天体验

#### 消息气泡

```
设计原则：
- 去气泡化: 相邻消息合并，只显示一次头像
- 时间戳: 悬停显示，避免视觉噪音
- 已读状态: ✓✓ 双勾，灰色→品牌色
- 编辑标记: "已编辑" 小标签

结构：
┌──────────────────────────────────────┐
│ [Avatar] [Username] [Timestamp]      │
│          Message content here...     │
│          [Reactions] [Actions]        │
└──────────────────────────────────────┘
```

#### 输入区域

```
设计原则：
- 焦点优先: 打开频道自动聚焦
- 富文本工具栏: 表情、附件、@提及、AI 助手
- 草稿自动保存: 离开保留内容
- 发送确认: Shift+Enter 换行，Enter 发送（可配置）

增强：
- AI 辅助: 输入时显示 AI 建议
- Slash 命令: / 开头触发命令面板
- 表情面板: 常用表情、搜索、GIF
```

### 3.8 错误处理

```
错误级别：
- Toast: 轻量提示，3s 自动消失
- Inline: 表单内联错误
- Modal: 严重错误，需要用户确认
- Page: 页面级错误（404、500）

文案风格：
- 清晰: 说清楚发生了什么
- 有助: 告诉用户能做什么
- 友好: 不责怪用户

示例：
❌ "请求失败"
✅ "网络不太顺畅，稍后再试试？"
```

### 4.4 功能优先级矩阵

```
                    用户价值
              低              高
        ┌─────────┬─────────┐
   低   │   P3    │   P2    │
  实    │  可做   │  应该做 │
  现    ├─────────┼─────────┤
  成    │   P4    │   P1   │
  本    │  不做   │  必须做 │
   高   └─────────┴─────────┘

**P0 必须做（低成本 + 高价值）**
- AI 分身基础能力
- 个人主页/动态名片
- 表情包生态增强
- 消息反应/特效

**P1 应该做（高成本 + 高价值）**
- 实时协作能力
- Live 房间
- 语音视频优化

**P2 可以做（低成本 + 中价值）**
- 积分系统
- 成就徽章
- 状态卡片

**P3 后续考虑（高成本 + 中价值）**
- AR 滤镜
- 故事功能

**P4 不做（低价值 + 高成本）**
- 过度游戏化功能
- 与核心定位冲突的功能
