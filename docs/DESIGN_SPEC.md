# Shadow 品牌与设计规范 v3.0

> **Version**: 3.0 | **Date**: 2024-03-24 | **Status**: Proposal

---

## 1. 品牌定位

**核心定位**: AI 原生的创作者社区平台

**目标用户**: 18-28 岁，内容创作者、独立开发者、AI 原住民

**品牌差异点**:
- AI Agent 是一等公民
- 创作者可变现
- 开源可定制

**Slogan**: `Your AI, Your Community, Your World`

---

## 2. 视觉风格

### 2.1 设计理念

**简洁、精致、有辨识度**

- 一个主色贯穿全局
- 克制使用强调色
- 留白与层次感

### 2.2 品牌色

**主色：Indigo 靛蓝**

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#6366F1` | 品牌识别 |
| 主色深 | `#4F46E5` | hover/active |
| 主色浅 | `#818CF8` | 次要元素 |

**中性色（暗色模式）**

| 用途 | 色值 |
|------|------|
| 背景 | `#0F0F10` |
| 卡片 | `#18181B` |
| 边框 | `#27272A` |
| 次文字 | `#71717A` |
| 主文字 | `#FAFAFA` |

**功能色**

| 用途 | 色值 |
|------|------|
| 成功 | `#22C55E` |
| 警告 | `#EAB308` |
| 错误 | `#EF4444` |

### 2.3 组件风格

- **按钮**: 主色填充，圆角，hover 微光
- **卡片**: 深色背景，细边框，hover 微浮
- **输入框**: 深色背景，focus 时主色边框
- **头像**: 圆形，带在线状态指示

### 2.4 动效

- 时长: 150-250ms
- 缓动: ease-out
- 场景: hover/active/focus/页面切换

### 3.3 个人主页
- 可定制 Profile
- 状态卡片
- AI 分身入口
