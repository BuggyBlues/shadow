# Shadow 品牌与设计规范 v2.0

> **Version**: 2.0 | **Date**: 2024-03-24 | **Status**: Proposal

---

## 1. 品牌定位

**核心定位**: AI 原生的创作者社区平台

**目标用户**: 18-28 岁，内容创作者、独立开发者、Web3/AI 原住民

**品牌差异点**:
- AI Agent 是一等公民（不只是机器人）
- 创作者可变现（内置电商/订阅）
- 开源可定制（vs Discord 闭源）

**Slogan 建议**: 
- 主推：`Your AI, Your Community, Your World`
- 备选：`Where AI Meets Community`

---

## 2. 视觉升级要点

### 2.1 色彩（核心变化）

| 当前 | 建议 | 原因 |
|------|------|------|
| 灰暗中性色 | 深邃紫 `#8B5CF6` + 霓虹粉 `#EC4899` | 更有活力、辨识度 |
| 扁平背景 | 微渐变 + 光晕效果 | 层次感、现代感 |
| 单一强调色 | 双色渐变品牌色 | 年轻化、视觉冲击 |

```css
/* 品牌渐变 */
--gradient-brand: linear-gradient(135deg, #8B5CF6 0%, #EC4899 100%);
```

### 2.2 组件风格

| 组件 | 当前 | 建议 |
|------|------|------|
| 按钮 | 纯色填充 | 渐变 + hover 光晕 |
| 卡片 | 扁平边框 | 微阴影 + hover 上浮 |
| 输入框 | 硬边框 | focus 时品牌色光晕 |
| 头像 | 圆形 | 保留，增加在线状态光圈 |

### 2.3 动效（关键场景）

- 按钮 hover: 缩放 0.98 + 光晕
- 页面切换: 滑入 250ms
- 消息发送: 弹性入场
- 尊重 `prefers-reduced-motion`

---

## 3. UI/UX 改进要点

### 3.1 消息体验
- 相邻消息合并头像（减少视觉噪音）
- 消息反应更突出（hover 显示，点击选择）
- 支持 Slash 命令 `/` 快捷操作

### 3.2 导航优化
- 服务器列表：图标优先，悬停显示名称
- 频道列表：未读用品牌色点标记
- 移动端：底部 Tab 导航 + 手势切换

### 3.3 个人主页（新功能）
- 可定制的 Profile 展示墙
- 状态卡片（在线/忙碌/自定义）
- AI 分身展示入口

## 4. 功能优先级

| 优先级 | 功能 | 价值 |
|--------|------|------|
| **P0** | 个人主页/状态卡片 | 自我表达是年轻用户刚需 |
| **P0** | 表情反应增强 | 降低表达门槛 |
| **P0** | 品牌色更新 | 视觉差异化第一步 |
| **P1** | AI 分身入口 | 核心差异化功能 |
| **P1** | 实时协作/白板 | 团队社区价值 |
| **P2** | 成就/积分系统 | 游戏化激励 |
